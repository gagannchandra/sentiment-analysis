**🟡 Currently under construction (aka 2/3 done and still fighting bugs like a warrior).**

---

## 🧠 Overview

This project focuses on building an end-to-end **Sentiment Analysis system** that classifies movie reviews as **positive or negative** using Natural Language Processing (NLP).

The goal?  
Compare **traditional machine learning** with **deep learning** and see who wins the battle.

(Spoiler: it depends.)

---

## ⚙️ Tech Stack

- **Python**
- **Scikit-learn**
- **TensorFlow / Keras**
- **Flask (API)**
- **Streamlit (UI)**

---

## 🔍 Features

### 🧹 Text Preprocessing Pipeline
- Tokenization
- Stopword Removal
- Lemmatization
- TF-IDF Vectorization

Basically, cleaning messy human opinions into something machines can understand.

---

### 🤖 Models Implemented

#### 1. Logistic Regression (Classic ML)
- Fast
- Interpretable
- Surprisingly strong baseline

#### 2. LSTM (Deep Learning)
- Captures sequence and context
- More powerful… but also more dramatic (needs tuning)

---

### 🌐 Deployment

- **Flask API** → for serving predictions  
- **Streamlit UI** → for real-time interaction  

Type a review → get judged instantly.

---

## 📊 Dataset

- IMDb Movie Reviews Dataset  
- Binary classification: **Positive / Negative**
